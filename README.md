# Guess The Number Game 🎮

A simple and interactive web-based game where the player has to guess a random number between 1 and 100.

## ✨ Features
* **Clean UI:** Minimalistic and responsive design.
* **Real-time Feedback:** Tells the user if the guess is "Too High", "Too Low", or "Correct".
* **Attempt Tracker:** Keeps track of how many tries the user took.

## 🛠️ Tech Stack
* **HTML5:** For the structure.
* **CSS3:** For styling and layout.
* **JavaScript:** For the game logic and DOM manipulation.
